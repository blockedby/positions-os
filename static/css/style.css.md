# style.css

Main dark theme stylesheet.

Contains:

- CSS variables (colors, spacing)
- Global reset and body styling
- Flexbox utilities (flex, h-screen, flex-1, overflow-auto)
- Spacing utilities (p-4, p-6, space-y-\*)
- Sidebar styling with navigation links
- Main content area styling
- Typography (h1, h2 headings)
- Component styles: cards, buttons, tables, status badges
